# Simple Client-Side Authentication

## Setup

1. **Clone the repository**:
```bash
git clone https://github.com/faran1385/login-task.git
```

2. **Install dependencies**:
```bash
npm install
```

## Run

1. **Start the development server:**:
```bash
npm run dev
```

2. **Open in browser:**:
```bash
http://localhost:3000
```
